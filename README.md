# 📄 Udon Thani Provincial Education Office PDF Helper (PEO PDF Helper)

ระบบจัดการไฟล์เอกสาร PDF บน Web Browser (Client-side 100%) ออกแบบมาเพื่อแก้ปัญหาและลดภาระงานด้านเอกสารของหน่วยงานราชการ โดยประมวลผลบนเครื่องของผู้ใช้งานโดยตรง ไฟล์เอกสารจะ **ไม่ถูกอัปโหลดขึ้นเซิร์ฟเวอร์** มั่นใจได้เรื่องความปลอดภัยของข้อมูล (Data Privacy)

## ✨ ฟีเจอร์ใหม่ล่าสุด (Latest Features)
- 💾 **Smart Remember Settings:** ระบบจดจำค่าการตั้งค่าล่าสุด (ขนาดฟอนต์, ตำแหน่ง, ระยะ Offset) ไว้ในเครื่องอัตโนมัติ เปิดใช้งานครั้งถัดไปไม่ต้องตั้งค่าใหม่
- 🧹 **Clear All Function:** ปุ่มล้างรายการหน้าเอกสารทั้งหมดในคลิกเดียว เพื่อเริ่มงานใหม่ได้ทันที
- 📱 **Mobile UI Optimization:** ปรับปรุงส่วนติดต่อผู้ใช้งานให้รองรับการทำงานบนสมาร์ทโฟนและแท็บเล็ตได้สมบูรณ์ยิ่งขึ้น

## 🛠️ ฟีเจอร์หลัก (Core Features)
- ➕ **Merge & Insert PDF:** รวมไฟล์ PDF หลายไฟล์และเลือกแทรกหน้าได้ตามต้องการ
- 🔄 **Rotate & Sort:** หมุนหน้าเอกสารและลากสลับตำแหน่ง (Drag & Drop) ได้อิสระ
- 🔢 **Smart Thai Numbering:** รันเลขหน้าอัตโนมัติ รองรับทั้ง **เลขไทย (๑, ๒, ๓)** และอารบิก
- 🎯 **Visual Offset:** ปรับระยะตัวเลขแกน X/Y ได้ละเอียด เพื่อหลบตราประทับหรือข้อความบนเอกสาร
- 🖨️ **Selective Printing:** เลือกเปิด-ปิดการพิมพ์เลขหน้าในบางหน้าได้ (เช่น หน้าปก) โดยที่ลำดับหน้ายังนับต่อเนื่องให้นะคะ

## 🚀 การติดตั้ง (Installation)
1. **Clone Repository:**
```bash
git clone https://github.com/mamazaki/peo-pdf-helper.git
```

2. ตรวจสอบโครงสร้างโฟลเดอร์:

- ฟอนต์สารบรรณต้องอยู่ที่ /fonts/THSARABUNNEW.TTF
- โลโก้หน่วยงานต้องอยู่ที่ /img/logo.png
- เปิดใช้งานผ่าน Web Server (Apache, Nginx หรือ GitHub Pages) เพื่อให้ระบบโหลดฟอนต์ผ่าน CORS ได้อย่างถูกต้อง
