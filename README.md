# 📄 Udon Thani Provincial Education Office PDF Helper

ระบบจัดการและรวมไฟล์ PDF ผ่านเว็บเบราว์เซอร์ พัฒนาขึ้นเพื่ออำนวยความสะดวกในการจัดทำเอกสารราชการของสำนักงานศึกษาธิการจังหวัดอุดรธานี เน้นการรันเลขหน้าด้วยฟอนต์มาตรฐานราชการไทย (TH Sarabun New) และการประมวลผลที่รวดเร็ว ปลอดภัย โดยทำงานฝั่ง Client-side 100%

## ✨ ฟีเจอร์หลัก (Features)

* **📑 รวมไฟล์ PDF (Merge PDF):** นำเข้าไฟล์ PDF หลายไฟล์เพื่อนำมาเรียงต่อกันได้อย่างอิสระ
* **🔢 ระบบรันเลขหน้าขั้นสูง:**
    * รองรับรูปแบบการรันลำดับ: อารบิก (1, 2, 3), เลขไทย (๑, ๒, ๓) และตัวอักษรไทย (ก, ข, ค)
    * รูปแบบการแสดงผลหลากหลาย เช่น `1`, `(1)`, `หน้า 1`, `- 1 -`, และ `หน้า 1 / 9`
    * สามารถเปิด-ปิด การรันเลขหน้าด้วย Master Toggle เพื่อใช้ระบบในการต่อไฟล์ PDF เพียงอย่างเดียวได้
* **🔤 รองรับฟอนต์ราชการไทยสมบูรณ์แบบ:** ฝัง (Embed) ฟอนต์ TH Sarabun New ครบทั้ง 4 สไตล์ (ปกติ, หนา, เอียง, หนาเอียง) ลงในเอกสาร PDF เพื่อให้แสดงผลถูกต้องในทุกอุปกรณ์
* **🎯 จัดตำแหน่งอิสระ (Precise Positioning):**
    * กำหนดตำแหน่งอัตโนมัติ: ขวาบน, กลางบน, ขวาล่าง, กลางล่าง
    * ปรับขยับตำแหน่ง (X / Y Offset) ได้อย่างละเอียด พร้อมเครื่องมือ UI ที่เข้าใจง่ายและระบบ Preview ดูการเปลี่ยนแปลงแบบ Real-time
* **🛠️ จัดการหน้าเอกสาร (Page Management):**
    * ลากและวาง (Drag & Drop) เพื่อสลับลำดับหน้าเอกสาร
    * หมุนหน้าเอกสาร (Rotate) และ ลบหน้า (Delete) เฉพาะหน้าที่ไม่ต้องการได้
    * ระบบ Zoom ดูรายละเอียดเนื้อหาหน้ากระดาษแบบความละเอียดสูง
* **🌗 Dark Mode:** รองรับการแสดงผลโหมดมืด เพื่อถนอมสายตา
* **🔒 ปลอดภัยและประหยัดทรัพยากร:** ประมวลผลข้อมูลและสร้างไฟล์ PDF ใหม่บนเครื่องของผู้ใช้ (Client-side Processing) ทั้งหมด ไม่มีเอกสารความลับถูกอัปโหลดขึ้นเซิร์ฟเวอร์ ลดภาระของ Hosting

## 🚀 เทคโนโลยีที่ใช้งาน (Tech Stack)

* **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
* **PDF Processing:** [pdf-lib](https://pdf-lib.js.org/) และ [pdf.js](https://mozilla.github.io/pdf.js/)
* **UI/UX:** [Sortable.js](https://sortablejs.github.io/Sortable/) (สำหรับระบบ Drag & Drop)

## 📦 การติดตั้ง (Deployment)

เนื่องจากระบบเป็นสถาปัตยกรรมแบบ Static Web Application สามารถนำไปวางบน Web Hosting หรือ Shared Hosting มาตรฐานทั่วไปได้ทันที โดยมีข้อกำหนดของโครงสร้าง Directory ดังนี้:

```text
/public_html
│── index.html
├── /img
│   └── logo.png
└── /fonts
    ├── THSARABUNNEW.TTF
    ├── THSARABUNNEWBOLD.TTF
    ├── THSARABUNNEWITALIC.TTF
    └── THSARABUNNEWBOLDITALIC.TTF
```
* (หมายเหตุ: ระบบทำงานโดยดึงไฟล์ฟอนต์จากโฟลเดอร์ /fonts โปรดตรวจสอบให้แน่ใจว่าชื่อไฟล์ตรงกันทุกตัวอักษร ทั้งตัวพิมพ์เล็กและตัวพิมพ์ใหญ่)

## 🚀 รายการอัพเดท (Release Notes / Change Log)
Version 1.2.0 (ล่าสุด)

✨ ฟีเจอร์ใหม่ (New Features)
   * ระบบ Drag & Drop: รองรับการลากไฟล์ PDF จากคอมพิวเตอร์มาวางในโปรแกรมเพื่อนำเข้าข้อมูลได้ทันที
   * ระบบตรวจจับไฟล์ที่มีรหัสผ่าน (Encryption Detection): เพิ่มความสามารถในการตรวจสอบไฟล์ PDF ที่มีการป้องกันด้วยรหัสผ่านก่อนนำเข้า
   * Custom Warning Modal: ออกแบบกล่องข้อความแจ้งเตือนใหม่ เพื่อแสดงขั้นตอนการแก้ไขปัญหาไฟล์ที่ล็อกรหัสผ่านให้ผู้ใช้งานทราบอย่างครบถ้วน (แทนที่การใช้ระบบ Alert มาตรฐานของ Browser)
* **🛠️ การปรับปรุงและแก้ไข (Improvements & Fixes)
   * ปรับค่าเริ่มต้นการรันเลขหน้า (Default Settings):
   * เปลี่ยนสถานะ "การรันเลขหน้าภาพรวม" เป็นปิด (Disabled) เป็นค่าเริ่มต้น
   * เปลี่ยนสถานะ "การพิมพ์เลขหน้า" ของแต่ละแผ่นงานเป็นซ่อนเลข (Unchecked) เป็นค่าเริ่มต้น
   * UI/UX Consistency: ปรับปรุงสีของ Label และสถานะปุ่มกดให้สอดคล้องกับค่าเริ่มต้นที่ถูกตั้งค่าไว้ เพื่อลดความสับสนของผู้ใช้งาน
   * Refactor Code: รวมศูนย์ฟังก์ชันการจัดการไฟล์ (File Handling Logic) เพื่อเพิ่มประสิทธิภาพและความเสถียรในการประมวลผล
* **🛡️ ความปลอดภัย (Security)
   * เพิ่มการรองรับ ignoreEncryption ในกระบวนการประมวลผลไฟล์ผ่าน pdf-lib เพื่ออนุญาตให้จัดการหน้าเอกสารที่มีการตั้งค่าสิทธิ์ (Permissions) พื้นฐานได้
